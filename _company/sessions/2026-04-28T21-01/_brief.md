# 📋 작업 브리프

**원 명령:** 내 유튜브 체널 분석해 지갑수호대

## 요약
사용자의 유튜브 채널 '지갑수호대'에 대한 성과 데이터를 수집하고, 이를 기반으로 초기 분석 보고서를 작성합니다.

## 분배
- **🔍 Researcher**: 시스템에 저장된 유튜브 API 키를 사용하여 '지갑수호대' 채널의 핵심 성과 지표(구독자 수, 조회수, 시청 시간 등)를 즉시 수집하고 데이터를 요약하라.
- **💰 Business**: 수집된 데이터를 분석하여 현재 채널의 성장 추세, 주요 콘텐츠의 성과, 그리고 잠재적인 수익화 기회에 대한 초기 비즈니스 통찰력을 도출하라.
- **📱 Secretary**: Researcher와 Business의 분석 결과를 바탕으로, 분석 보고서의 초안을 작성하고 다음 단계의 전략적 제안을 준비하라.
